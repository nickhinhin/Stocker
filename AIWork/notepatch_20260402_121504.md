# Stocker 本輪改善紀錄（自動化）

## 本輪完成
- UI：加強 Liquid Glass 風格（半透明面板、更柔和陰影、玻璃模糊層次），並為新模組補上對應樣式。
- 圖表：新增 `Portfolio State Trend (Assets / Cash / Profit)`，可同時觀察總資產、現金與總收益趨勢。
- 功能：新增 `Load Regression Test Portfolio`（選擇頁 + Data 頁），一鍵生成回歸測試投資組合（多市場、多幣別、多交易型態、跨年度）。
- 測試資料：內建 2 組測試組合（US Growth / Asia Income），涵蓋 BUY/SELL/DIVIDEND_CASH/CASH/CASH_CONVERT/FEE/INTEREST。
- Bug 檢查：新增交易資料品質檢查面板（Transactions 頁），可檢出：
  - 缺少 symbol/currency
  - 非法股數（<=0）
  - 負數價格或費用
  - 未來日期
  - 疑似重複交易
  - 賣出超過歷史持股
- 快速修正：每條問題可直接點 `Edit` 跳交易編輯。

## 驗證結果
- TypeScript：`node node_modules/typescript/bin/tsc -p tsconfig.json` 通過。
- Build：`npm run build` 仍受環境限制失敗（`esbuild spawn EPERM`），屬於本機執行權限問題，不是這輪邏輯改動造成。
